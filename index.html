📱 次から“アイコン”で一発起動する方法

毎回URLを探さなくてOK。ホーム画面に追加すると、アプリみたいにすぐ開けます。

iPhone の方（Safari）
  1. このページ or 開きたいアプリを Safari で開く
  2. 画面下の 共有ボタン（□↑） をタップ
  3. 「ホーム画面に追加」 をタップ
  4. 右上の 「追加」 を押して完了
Android の方（Chrome）
  1. このページ or 開きたいアプリを Chrome で開く
  2. 右上の ︙（3点） をタップ
  3. 「ホーム画面に追加」 をタップ
  4. 「追加」 を押して完了
LINEから開くときのコツ（うまく開けない時）
  1. LINEの青いURLをタップして開く
  2. 画面の ︙ / … をタップ
  3. 「ブラウザで開く（Safari/Chrome）」 を選ぶ
  4. その後に ホーム画面に追加 を行うと確実です
